# AUMS Course Booster

AUMS Course Booster is a lightweight browser extension designed to enhance your experience on the Amrita Academic Suite (AUMS). 

## Features

- **Theme Customization**: Switch between multiple themes to suit your viewing preferences. Supported themes include:
  - Dark Theme
  - Light Theme
  - Dracula Theme
- **Quick File Access**: Instantly list and collect URLs for all downloadable course materials (PDFs, Word documents, Excel sheets, and PowerPoint presentations) on the current page.

## Installation

1. Clone or download this repository to your local machine.
2. Open your Chromium-based browser (Chrome, Edge, Brave, etc.) and navigate to the extensions page (`chrome://extensions/` or `edge://extensions/`).
3. Enable **Developer mode**.
4. Click on **Load unpacked** and select the directory containing the extension files.
5. The AUMS Course Booster icon should now appear in your browser's toolbar.

## Usage

- Navigate to the Amrita Academic Suite (`https://aumscb.amrita.edu/*`).
- Click the extension icon in your toolbar to open the popup interface.
- Use the provided buttons to toggle themes or list available files for download.

## Permissions

This extension requires the following permissions to function correctly:
- `scripting`: To apply themes and extract file links from the page.
- `storage`: To save your preferred theme locally.
- `activeTab`: To interact with the currently active AUMS page.
